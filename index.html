Welcome!

This website will help you revise key GCSE Computing topics including hardware, software, and practice questions.

Hardware

  • CPU: Executes instructions, the brain of the computer.
  • RAM: Temporary memory used while programs run.
  • Storage: Hard drives and SSDs store data permanently.

Software

  • System Software: Operating systems manage hardware and software.
  • Application Software: Programs like word processors and games.
  • Utility Software: Tools for maintenance, e.g., antivirus.

Practice Questions

  1. What does CPU stand for?
  2. Give one example of application software.
  3. What is the difference between RAM and storage?
